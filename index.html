As such, it is entirely constructed using HTML, CSS, and JavaScript. It can also be used as a refresher for basic uses and operations of several common Linux terminal commands.

cd - used to change directories
sudo - preface a command with this to grant it root privileges
pwd - output current directory location
ls - list contents of directory: 'ls -l' gives more details and 'ls -a' shows hidden files
man - display reference manual page for command in the terminal
chmod - modify permissions of file: 'sudo chmod 777 FILE' grants full access to the specified file for all users
rm - remove file or directory: use 'rm -r' to remove directories and their contents
dd - copy contents of input onto output, converting/formatting as specified: useful for data recovery and OS flashing
nc - used to initiate netcat connection
ssh - establish a secure connection to a remote machine, through which commands can be executed
grep - search through input, outputting only specified patterns
find - search for file in specified directory hierarchy matching listed arguements
echo - output specified text
cat - output contents of file into terminal
ps aux - outputs a snapshot of the processes for every user, along with useful information pertaining to each process
top - output and monitor the processes using the most system resources in the terminal

Useful Tricks: Can rerun a command by using '!!', especially useful for rerunning a command as root.
Can remove specified pattern from output stream by using >/dev/null to pipe it into a "black hole".
